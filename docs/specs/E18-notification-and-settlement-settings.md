# E18 알림 및 정산 설정 스펙

- 작성일: 2026-03-18
- 대상 작업: `E18-3`
- 관련 API: `GET/PUT /api/users/me/notification-preferences`, `GET/PUT /api/users/me/settlement-settings`
- 관련 화면: `frontend/src/pages/Profile.jsx`

## 목적
- 사용자가 주요 외부 알림 수신 여부와 정산 기본 안내 문구를 프로필 설정 화면에서 직접 관리할 수 있게 한다.
- 반복해서 쓰는 정산 안내를 개인 기본값으로 저장해 이후 운영 흐름에서 재사용할 기반을 만든다.

## 알림 설정
- 기존 알림 타입별 브라우저 푸시 토글을 유지한다.
- 지원되는 브라우저 푸시 타입에 한해 `브라우저 푸시 전체 받기` 토글을 추가한다.
- 전체 토글은 지원 대상 알림 타입을 모두 같은 값으로 저장한다.

## 정산 설정
- `정산 기본 안내` 텍스트 필드를 추가한다.
- 최대 길이는 300자다.
- 비워두면 기본 안내가 없는 상태로 저장한다.

## API 응답 예시
```json
{
  "settlementGuide": "입금 후 채팅에 마지막 4자리를 남겨주세요."
}
```

## 화면 동작
- 알림 설정 카드
  - 현재 브라우저 연결/해제
  - 브라우저 푸시 전체 토글
  - 알림 타입별 개별 토글
- 정산 기본 안내 카드
  - 다중행 입력
  - 글자 수 표시
  - 저장 버튼

## 제외 사항
- 앱 내 알림 자체를 완전히 끄는 기능은 이번 카드 범위에 포함하지 않는다.
- 정산 기본 안내를 파티 상세나 채팅 공지에 자동 삽입하는 기능은 후속 카드에서 연결한다.
